# sci-fi-ui
Sci-fi based UI made in Processing 3  
Made for my second year semester 1 Programming assignment.

### Requirements
* JDK 
* Processing 3 editor

### How to play
* to speed up press W
* to slow down press S
* to look left press A
* to look right press D
* all other control is done using the mouse
